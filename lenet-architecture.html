Layer Details

📥 Input Layer (32×32)

Raw grayscale images of handwritten digits. Each pixel value normalized to [0,1] range.

🔵 C1 - Conv Layer

6 filters of size 5×5, produces 6 feature maps of 28×28. Each filter extracts local features.

🟠 S2 - Subsampling

2×2 average pooling with stride 2. Reduces spatial dimensions while retaining important features.

🟣 C3 - Conv Layer

16 filters of size 5×5, connected to all 6 S2 maps. Creates more complex feature combinations.

🟠 S4 - Subsampling

2×2 average pooling with stride 2. Further reduces dimensions to 5×5 for 16 maps.

🟢 C5 - Fully Connected

120 neurons, each connected to all 16 S4 feature maps. Flattens features for classification.

🟢 F6 - Fully Connected

84 neurons in the hidden layer. Uses tanh activation function.

🔴 Output Layer (10)

Softmax classifier for digits 0-9. Outputs probability distribution over 10 classes.
